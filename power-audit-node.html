Smart Energy

Power Audit Node

Distributed IoT nodes that watch voltage usage across lab equipment and send alerts when power draw looks unusual.

Overview

The Power Audit Node was built to help teams understand how their lab devices behave over time. Each node measures voltage patterns, packages the readings, and sends them to a central dashboard over MQTT. The goal is not just monitoring; it is making abnormal behavior visible early enough that a team can react before a device fails in the middle of a session.

We kept the hardware simple so that deployment would be practical. An ESP32 handles connectivity, while a sensing circuit tracks voltage changes with enough precision to spot meaningful trends. The firmware batches readings, filters out obvious noise, and reports only the information that matters for day-to-day use.

In testing, the most useful feature turned out to be the alert threshold tuning. Different lab tools have different normal signatures, so the node can be adjusted per device. That flexibility makes the system usable across benches instead of only in one controlled setup.
